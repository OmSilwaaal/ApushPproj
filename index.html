    ABOUT THIS PROJECT

    Vietnam: The Long War

    An AP US History strategic simulation covering the Vietnam War from 1963–1975.

    LSTM Neural Network

    The enemy AI uses a Long Short-Term Memory (LSTM) recurrent neural network that learns from every action you take. Each game event (artillery strike, recon deployment, troop movement) becomes a node in the network. The AI adapts its tactics across battles.

    Ablation Analysis

    After each battle, the LSTM performs an ablation study — systematically removing each input variable to determine its contribution to the outcome. This shows which decisions actually mattered.

    Historical Accuracy

    All 8 battles are in true historical order. Events, unit capabilities, terrain, and political impacts are based on historical records.

    Public Opinion System

    Your choices affect domestic support. Civilian casualties, use of chemical weapons, and costly battles erode public opinion. When thresholds are crossed, real historical anti-war events are triggered.
